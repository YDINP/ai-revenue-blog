## n8n이란?

n8n(노드에이트엔)은 **오픈소스 워크플로우 자동화 도구**입니다. Zapier나 Make(구 Integromat)와 비슷하지만, 셀프 호스팅이 가능하여 **무제한 무료 사용**이 가능합니다.

## 왜 n8n으로 수익화인가?

| 비교 | Zapier | Make | n8n (Self-hosted) |
|------|--------|------|------------------|
| 가격 | $19.99/월~ | $9/월~ | **무료** |
| 실행 제한 | 750/월 (무료) | 1,000/월 (무료) | **무제한** |
| 커스텀 코드 | 제한적 | 제한적 | **완전 자유** |
| AI 연동 | 가능 | 가능 | **완전 자유** |

## 자동 수익 파이프라인 3가지

### 1. AI 블로그 자동 발행 파이프라인

```
매일 아침 9시 (Cron 트리거)
    ↓
트렌드 키워드 수집 (EXA/Google Trends API)
    ↓
AI로 SEO 최적화 글 생성 (Claude/GPT API)
    ↓
제휴 마케팅 링크 자동 삽입
    ↓
GitHub에 커밋 (정적 사이트 자동 배포)
    ↓
소셜 미디어 자동 공유
```

**예상 수익**: 쿠팡파트너스 + AdSense로 월 5~30만원

### 2. 뉴스레터 자동 큐레이션

```
매일 저녁 (Cron 트리거)
    ↓
RSS/API로 관련 뉴스 수집
    ↓
AI로 요약 및 큐레이션
    ↓
이메일 서비스 API로 발송
    ↓
오픈율/클릭률 추적
```

**예상 수익**: 스폰서십으로 월 10~50만원 (구독자 1,000명+)

### 3. 디지털 상품 자동 생성

```
주 1회 (Cron 트리거)
    ↓
시장 트렌드 분석
    ↓
AI로 디지털 상품 생성 (템플릿, 가이드, 프롬프트)
    ↓
Gumroad/크몽에 자동 등록
    ↓
판매 알림 수신
```

**예상 수익**: 월 5~20만원

## 시작하기

### 1단계: n8n 설치

```bash
npm install -g n8n
n8n start
```

브라우저에서 `http://localhost:5678`로 접속합니다.

### 2단계: 첫 워크플로우 만들기

1. **Schedule Trigger** 노드 추가 (매일 09:00 설정)
2. **HTTP Request** 노드로 트렌드 API 호출
3. **AI Agent** 노드로 콘텐츠 생성
4. **GitHub** 노드로 파일 커밋

### 3단계: 모니터링 설정

n8n의 실행 로그에서 성공/실패를 추적하고, 에러 발생 시 이메일 알림을 설정하세요.

## 비용 분석

| 항목 | 월 비용 |
|------|---------|
| n8n (셀프 호스팅) | 0원 |
| AI API (Haiku) | ~$5 (약 7,000원) |
| 도메인 | ~$1 (약 1,400원) |
| **총 비용** | **약 8,400원/월** |

**손익분기점**: 월 1만원 이상 수익이면 흑자!

## 주의사항

- AI 생성 콘텐츠는 반드시 **품질 검수 로직**을 추가하세요
- 쿠팡파트너스 링크는 **대가성 문구**를 반드시 표시해야 합니다
- 소셜 미디어 자동 포스팅은 **스팸 정책**을 준수하세요

## 결론

n8n + AI API 조합은 **가장 낮은 진입장벽**으로 자동 수익 파이프라인을 구축할 수 있는 방법입니다. 셀프 호스팅으로 비용을 최소화하면서, AI의 콘텐츠 생성 능력을 최대한 활용할 수 있습니다.

생산성을 높이려면 좋은 장비도 중요합니다. 특히 여러 워크플로우를 동시에 돌리려면 충분한 RAM과 프로세서 성능이 필요하죠.
